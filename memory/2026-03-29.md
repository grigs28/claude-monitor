### 使用方法

```bash
# 1. 启动 Claude Code
tmux new-session -s claude
conda activate taf
cd /mnt/SDD/webapp/TAF
claude

# 2. 启动自动确认器
python3 /mnt/SDD/opt/claude-monitor/smart_confirmer.py claude

# 3. 查看当前模型
python3 /mnt/SDD/opt/claude-monitor/switch_model.py
```
